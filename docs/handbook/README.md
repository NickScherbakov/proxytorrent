# Справочник проекта ProxyTorrent

## О справочнике

Этот справочник содержит полное описание проекта ProxyTorrent — комбинированного сервиса прокси и BitTorrent, который безопасно загружает контент через изолированный прокси/VPN, упаковывает его в торренты и раздаёт запрашивающим клиентам.

**Цель справочника:** предоставить разработчикам, администраторам и аудиторам единый источник информации о целях, архитектуре, безопасности, развёртывании и результатах проекта.

## Как читать справочник

Справочник организован в виде последовательных глав. Новым участникам рекомендуется читать по порядку, начиная с миссии проекта и заканчивая известными ограничениями. Опытным пользователям можно переходить к нужным разделам напрямую.

### Для разных ролей

- **Разработчики**: главы 3-7, 10, 12
- **DevOps/Администраторы**: главы 6, 8, 9, 10
- **Архитекторы**: главы 2, 3, 4, 7, 8
- **Руководители проекта**: главы 1, 2, 11, 12
- **Аудиторы безопасности**: главы 4, 6, 8
